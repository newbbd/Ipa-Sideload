Certificates

Use this as a signing guide, not a cert market.

Shared certificates and random enterprise profiles can be revoked, abused, or used to trick people into installing unsafe apps. This site should explain safe signing routes instead of distributing cert files.

Recommended signing categories

  • Free Apple ID sideloading with refresh limits
  • Paid Apple Developer Program signing
  • TestFlight beta distribution
  • Region-supported marketplace options where legally available

Rules

Safety checklist

Use these rules to keep the directory useful and clean.

01

No cracked IPAs

Do not list pirated apps, paid app dumps, game cheats, or modified apps made to bypass payment.

02

No leaked certs

Do not host or promote random P12 files, enterprise certs, or Telegram certificate drops.

03

Official links first

Prefer the developer website, verified GitHub repo, official docs, or app marketplace page.

04

Be honest about risk

Label tools that need a computer, refresh apps, use profiles, or depend on region-specific rules.
